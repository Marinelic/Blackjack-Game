A simple, interactive Blackjack game built with HTML, CSS, and vanilla JavaScript.
This project was made to practice JavaScript fundamentals such as arrays, conditionals, DOM manipulation, and game logic.



FEATURES
- Draw cards and see your hand update live
- Dealer receives cards automatically
- Blackjack detection
- Bust and game-over logic
- Randomized chip amount for the player
- Clean, modern UI with dealer & player sections


Live Demo - https://blackjack-game-bice.vercel.app/
